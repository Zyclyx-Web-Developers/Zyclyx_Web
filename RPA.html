Overview

Process Automation using robotics or artificial intelligence provides an opportunity to automate rules-based processes that are still done by humansThe term ‘Robotic Process Automation’ (RPA) is used to refer to this fast-emerging category of business process software that can be rapidly customized to run existing enterprise, web, desktop and mainframe applications in the same way teams of people do.
